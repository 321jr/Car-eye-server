# car-eye-server
car-eye服务器主要用于发布car-eye管理系统服务器方面的源码和设计
car-eye 服务器主要包含车辆设备管理平台,采用最先进的架构，实现了对车辆管理，车辆监控，车辆数据统计等功能，是car-eye车辆管理系统的中心.
